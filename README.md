# The Atlas Engine

The Atlas Engine is a side-project of mine aiming to create a game engine that is flexible, easy to use and fast, scraping out the bloat of other engines. It is written in C++ and uses GLFW for window management and OpenGL for rendering.
I am currently working on the engine in my free time, so it is not yet complete. However, I am making good progress and hope to have a working prototype soon. I will be also working on Metal and Vulkan backends in the future to make rendering
stunning graphics possible on all platforms.
