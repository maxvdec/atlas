# Atlas Roadmap

This is a sketched roadmap, the full roadmap is in the website.

## Roadmap to Beta

- Basic window creation and management
- Input handling (keyboard, mouse)
- Basic rendering (shapes, textures)
- Simple UI
- Editor UI
- Editor application
- Editor runtime
- Add mesh loading
- Scene management
- Physics engine integration (Bezel)
- Audio support (Finewave)
- Scripting support
- Asset management
- Documentation and examples
- Particle system
- Lights
- Shadows (except for point lights)
- Post-processing effects
- Sky, atmosphere, weather and fluid simulation system (Hydra)
- Water rendering system
- Weather system
- Cloud system
- Screen-space global illumination
- Night-day cycle system
- Add sun, moon and stars rendering
- Animation system (Mold)
- Instancing support
- Optimizations and performance improvements
- CLI for packing applications
- Runtime independent of final executable
- Scene packing
- Add deferred rendering for better performance with many lights
- Add PBR rendering
- Add other texture supports (normal, roughness, metallic, ao, etc.)
- Add HDR support
- Add SSAO
- Add soft shadows
- Add shadows for point lights
- Add normal and parallax mapping
- Add volumetric lighting
- Add bloom for lights
- Add depth of field
- Add terrain system (Aurora)
- Add motion blur
- Physically based bloom
- Add area lights
- Screen-space reflections
- Add rim lighting
- Add chromatic aberration
- Add posterization
- Add pixelization
- Add dilation
- Add film grain
- Add LUT tables
- Add foam and simple fluid rendering
- Screen-space refractions
- Create opal for core rendering
- Migrate the engine to use opal
- Add Vulkan support through opal
- Add Jolt Physics instead of Bezel
- Better website
- More modern documentation
- Examples and tutorials
- Create the Hana shader language
- Create syntax highlighting for Hana
- Move shaders to hana

## Other features planned for the future

- AI system (Adverse)
- UI library (Graphite)
- Vegetation and foliage system
- Terrain generation and rendering (Terra)
- Network support (Lasse)
- Metal support
- Ray tracing rendering (Overflow)
- Add 2d physics (Bezel 2D support)
