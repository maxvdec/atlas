Detailed Description

Component that provides audio playback capabilities to an object. It can be attach to any object in the scene and allows for playing, pausing and stopping audio.

Note
Finewave will be required for this component to work. This component under the hood uses Finewave's AudioSource class to manage audio playback.

Example

// Create an AudioPlayer component
auto audioPlayer = std::make_shared<AudioPlayer>();
// Initialize the audio player
audioPlayer->init();
// Set the audio source from a file
audioPlayer->setSource(Workspace::get().getResource("AudioResource"));
// Play the audio
audioPlayer->play();
