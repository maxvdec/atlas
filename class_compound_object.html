Detailed Description

A Compound object is a GameObject that can be extended by the user to generate objects that encapsulate multiple CoreObject. This is useful for creating complex objects that are made up of multiple simpler objects.

Example of a compound object

class Car : public CompoundObject {
public:
void init() override {
// Create and add car body
CoreObject *body = new CoreObject();
// Set up body properties...
addObject(body);
}
void updateObjects(Window &window) override {
// Update car-specific logic, e.g., movement
}
};
