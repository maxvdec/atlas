Detailed Description

UI element that renders a textured rectangle on screen.

Example

Image logo(Texture::fromResource(resource),
{.width = 160.0f, .height = 160.0f},
{.x = 32.0f, .y = 32.0f});
logo.tint = Color::white();
logo.style().normal().cornerRadius(18.0f);
