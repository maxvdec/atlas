Detailed Description

TCP client that connects to a server and streams newline-delimited messages.

The pipe owns a background receive thread. Use start() / stop() to control the connection loop.

Example

NetworkPipe pipe;
pipe.setPort(5123);
pipe.onRecieve([](const std::string& msg) {
std::cout << "Tracer: " << msg << std::endl;
});
pipe.start();
pipe.send("hello\n");
// ... later
pipe.stop();
