Detailed Description

Tessellated terrain surface that can be generated procedurally or from heightmaps.

Example

Terrain terrain(HillGenerator());
terrain.width = 512;
terrain.height = 512;
terrain.initialize();
terrain.addBiome(Biome("Snow", Color::white()));
window.addObject(&terrain);
