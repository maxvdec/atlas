33/**
34 * @brief TCP client that connects to a server and streams newline-delimited
35 * messages.
36 *
37 * The pipe owns a background receive thread. Use `start()` / `stop()` to
38 * control the connection loop.
39 *
40 * \subsection networkpipe-example Example
41 * ```cpp
42 * NetworkPipe pipe;
43 * pipe.setPort(5123);
44 * pipe.onRecieve([](const std::string& msg) {
45 * std::cout << "Tracer: " << msg << std::endl;
46 * });
47 * pipe.start();
48 *
49 * pipe.send("hello\n");
50 * // ... later
51 * pipe.stop();
52 * ```
53 */
