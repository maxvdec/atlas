Detailed Description

Structure representing a single vertex in 3D space, including position, color, texture coordinates, and normal vector.

Warning
This is meant for internal use only and should not be used directly because it's difficult to read and maintain.

Example

// Create a vertex at position (1, 2, 3) with red color
CoreVertex vertex({1.0, 2.0, 3.0}, {1.0, 0.0, 0.0, 1.0}, {0.5, 0.5}, {0.0,
0.0, 1.0});
