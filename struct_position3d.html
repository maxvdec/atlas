Detailed Description

Structure representing a position in 3D space with double precision. Provides arithmetic operations and conversions to/from GLM types.

Example

// Create a position
Position3d pos(10.0f, 5.0f, -3.0f);
// Move it
pos += Position3d(1.0f, 0.0f, 0.0f);
// Convert to GLM vector for rendering
glm::vec3 glmPos = pos.toGlm();
